HttpRequest Class Reference

Detailed Description

An HTTP request that servers can read from.

An HttpRequest has a method (such as GET or POST), optional headers, and optional body. The body can be a String, an Array<byte> or a Var. In the case of a Var it will be encoded as JSON and the request content type header automatically set.

Math

Detailed Description

Classes and functions for math, including types useful in 2D/3D geometric computations and graphics.

These classes are templates that can be used with elements of any numeric type.

Vec3_<int> ivec(1, 2, 3);
Matrix4_<float> m;

They all have two predefined specializations, for float and double elements. The type for doubles has a d suffix.

Vec3 v; // Vec3_<float>
Matrix4; // Matrix4_<float>
Vec3d v; // Vec3_<double>
Matrix3d; // Matrix3_<double>
Matrixd; // Matrix_<double>

Matrix_<T> asl::solveZero ( F  f,
const Matrix_< T > &  x0 
)

Solves a system of equations F(x)=[0], given by functor f, which returns a vector of function values for an input vector x; and using x0 as initial guess.

If there are more equations than unknowns (f larger than x0) then a least-squares solution is attempted.

Example:

Solve for (x, y) in:

  • (x-1)2 + (y-1)2 - 1 = 0
  • sin(x) + sin(y) - x = 0
auto x = solveZero([](const Matrixd& x) { // unknowns are x[0], x[1]
return Matrixd{
sqr(x[0] - 1) + sqr(x[1] - 1) - 1,
sin(x[0]) + sin(x[1]) - x[0]
};
}, { 0.0, 0.0 }); // initial estimation

Detailed Description

template<class T>
class asl::Vec2_< T >

A Vec2 represents a vector in 2D space.

This class allows operating with vectors as with primitive types via operators.

Vec2 a (10, 10);
Vec2 b (-5, 1.5);
Vec2 c = -a.rotate(PI/4) + 2.0f * (b - a).normalized();

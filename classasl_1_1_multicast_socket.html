MulticastSocket Class Reference
Socket communication

Detailed Description

A MulticastSocket is a communication socket for multicast UDP protocol.

It has to use class D addresses (224.0.1.0 - 239.0.0.0) which identify multicast groups.

InetAddress group("224.0.1.1", 18000);
asl::InetAddress
Represents an endpoint of a socket which is ususally an IP address and a port.

A listening side would join the group, receive packets and then leave:

MulticastSocket socket;
socket.join(group);
InetAddress sender;
socket.readFrom(sender, data);
socket.leave(group);

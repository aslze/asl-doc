StreamBuffer Class Reference
Binary data

Detailed Description

This class is a buffer that can be written to as a binary stream.

The buffer is initially empty and grows as you append variables. You can change endianness at any moment.

You can then get the content as an Array<byte> and for example write it to a file or send it through a socket.

StreamBuffer buffer(ENDIAN_BIG);
buffer << 1 << "abc" << 1.5 << short(33);
File("data").put(buffer);
socket << *buffer;

#include <StreamBuffer.h>

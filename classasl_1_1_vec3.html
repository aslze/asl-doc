Vec3 Class Reference

Detailed Description

Vec3 represents a vector in 3D space.

This class allows operating with vectors as with primitive types via operators. It can be used together with class Matrix4 to transform vectors in space.

Vec3 a (10, 10, 0.1);
Vec3 b (-5, 1.5, 50);
Vec3 c = -a + 2.0 * (b - a).normalized();

The angle between two vectors can be computed with:

float angle = acos(a/!a * b/!b);

Random Class Reference

Detailed Description

A random number generator.

Generates uniformly distributed pseudo-random numbers, except in the normal() functions.

Random random;
int n = random(256); // get an integer between 0 and 255
double x = random(-1.5, 1.5); // get a number between -1.5 and +1.5
double y = random.normal(10, 0.75); // get a number between -1.5 and +1.5

The generator initially has a constant seed, so it will always produce the same sequence. You can change the seed with the init() function, either with values or with no arguments to automatically seed it randomly:

random.init();

Or you can create the object with a true argument to auto seed it :

Random random(true);

For compatibility with older code, there is a global asl::random object already random initialized ready for use. But it is recommended to use new Random objects when separate sequences or multithreading are needed.
